# Shawarma Empire

## Project

This is a commercial idle/tycoon Android game built with Godot 4.

## Main goals

- Clean architecture.
- Small, reviewable changes.
- Production-quality code.
- Easy to maintain.
- Mobile-first.
- 60 FPS on low-end Android devices.

## Coding rules

- Use GDScript 2.
- Use static typing whenever possible.
- Never duplicate code.
- Keep functions short.
- Use descriptive variable names.
- Separate UI from gameplay logic.
- Avoid magic numbers.
- Use constants.
- Create reusable components.
- Never rewrite existing systems unless necessary.

## Project structure

Scenes/
Scripts/
Assets/
Resources/
Audio/
UI/
Managers/

## Every task

Before changing code:
- Understand existing architecture.

After changing code:
- Ensure project has no parser errors.
- Explain what changed.
- Keep commits small.code
